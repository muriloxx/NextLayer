<h2>Sobre o Projeto (PIM IV)</h2>

<p>
  Este repositório contém o desenvolvimento do <strong>Projeto Integrado Multidisciplinar (PIM) IV</strong> do curso de <strong>CST em Análise e Desenvolvimento de Sistemas</strong> da UNIP. O projeto é uma continuação do PIM III (onde foi realizado o levantamento de requisitos) e foca na implementação de um sistema de suporte técnico inteligente.
</p>

<p>
  O objetivo geral é <strong>desenvolver um sistema de suporte técnico inteligente</strong>, que utiliza recursos como FAQs dinâmicas (baseadas no histórico de chamados) e ferramentas de IA para otimizar o atendimento e promover a autonomia dos usuários. Todo o tratamento de dados pessoais no sistema seguirá as diretrizes da <strong>LGPD</strong>.
</p>

<h2>Cenário Proposto</h2>

<p>
  O sistema foi projetado para uma empresa de médio porte que atualmente gerencia seu suporte técnico interno via e-mail e telefone. Este método atual causa dificuldades no controle, atrasos e falhas na priorização.
</p>
<p>
  A solução (NextLayer) é um sistema integrado onde os colaboradores podem registrar suas solicitações. A Inteligência Artificial auxiliará sugerindo soluções automáticas ou encaminhando o chamado ao técnico mais adequado, com base no histórico e complexidade do problema.
</p>

<h2>Tecnologias Utilizadas</h2>

<p>Conforme as diretrizes do projeto, as seguintes tecnologias são a base deste sistema:</p>

<ul>
  <li><strong>Backend (Web):</strong> ASP.NET e C#</li>
  <li><strong>Frontend (Desktop):</strong> C# com Windows Forms, WPF ou Blazor</li>
  <li><strong>Mobile:</strong> Desenvolvimento para Android</li>
  <li><strong>Banco de Dados:</strong> Postgres</li>
  <li><strong>Inteligência Artificial:</strong> Uso de ferramentas de IA generativa para apoiar a criação de FAQs, sugestões de respostas automáticas e documentação técnica.</li>
</ul>
